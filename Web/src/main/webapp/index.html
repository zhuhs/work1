  • place通用方法

    在目标浏览器上点击右键-->属性，打开的对话框中找到【目标】一栏

    选择【目标】右键【复制】，复制下IE的安装路径

    回到桌面，在空白的地方右键，选择【新建】，然后选择【快捷方式】

    在弹出的对话框内右键【粘贴】刚才复制的路径，记得后面加一个空格，后面贴上工具网址https://www.pomit.cn/desktop.html
